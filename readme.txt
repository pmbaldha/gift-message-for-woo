== Installation ==

= Automatic Installation (Recommended) =

1. Go to **Plugins ‚Üí Add New** in your WooCommerce admin
2. Search for "**Gift Message for Woo**" or "**WooCommerce Gift**"
3. Click **Install Now** on our gift message plugin
4. Activate the gift plugin for WooCommerce
5. Gift messages are now enabled in your Woo store!

= Manual Installation for WooCommerce =

1. Download the Gift Message for Woo plugin
2. Upload to `/wp-content/plugins/gift-message-for-woo/`
3. Activate through WooCommerce Plugins menu
4. Gift functionality is ready in your Woo store

= After Installing Gift Message for WooCommerce =

* Visit any WooCommerce product to see the gift message field
* Test gift notes in your Woo cart and checkout
* Check WooCommerce orders for gift message column
* Configure gift settings using Woo hooks

== Frequently Asked Questions ==

= Does this gift plugin work with all WooCommerce themes? =

Yes! Gift Message for Woo works with any properly coded WooCommerce theme. The gift fields use standard Woo hooks for maximum compatibility.

= Can I customize the gift message appearance in WooCommerce? =

Absolutely! The gift message fields include CSS classes for styling in your Woo theme. Use hooks to customize gift labels, limits, and validation in WooCommerce.

= Are gift messages included in WooCommerce emails? =

Yes! Gift notes appear in all WooCommerce order emails - perfect for gift purchases. Both customers and admins see gift messages in Woo notifications.

= Does this support WooCommerce variable products for gifts? =

Currently supporting simple products in WooCommerce. Variable product gift messages coming soon to Woo!

= Can I change the gift message character limit in WooCommerce? =

Default is 150 characters for gift notes in Woo. Use our hooks to customize gift limits per product or category in WooCommerce.

= Is this compatible with WooCommerce Blocks and Woo checkout? =

Yes! Full compatibility with WooCommerce Blocks, classic Woo checkout, and modern gift shopping flows.

= Does the gift plugin work with WooCommerce HPOS? =

100% compatible with WooCommerce High-Performance Order Storage for gift messages.

= Can customers edit gift messages after ordering in WooCommerce? =

Gift messages can be edited in Woo cart/checkout. Post-order gift edits require WooCommerce admin access.

= How do gift messages appear in WooCommerce admin? =

Gift notes show in a dedicated column in WooCommerce orders list. Full gift messages visible in Woo order details.

= Is this gift plugin translatable for WooCommerce stores? =

Yes! Gift Message for Woo is translation-ready for international WooCommerce stores.
